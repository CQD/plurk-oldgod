# 噗浪老神

[噗浪](https://plurk.com)上的[老神](https://www.plurk.com/OldGod)機器人的原始程式碼，host 在 Google App Engine 上。

邏輯用 PHP 實作。原先只是個好玩的專案，不知不覺就放著跑了好幾年。

## 機器人使用說明

若想使用老神，請先將老神（[OldGod](https://www.plurk.com/OldGod)）這個帳號加入好友。然後發噗時以「老神」開頭即可。

目前支援的功能如下：

求籤
- 「老神我想打籃球，請賜籤」
- 「老神 求籤 明天的天氣」
- 「老神 抽籤問腳踏兩條船的運勢」

問吉兇：
- 「老神 請問 ONS 的吉兇」
- 「老神 問吉兇 打小孩」
- 「老神 我要後宮」
